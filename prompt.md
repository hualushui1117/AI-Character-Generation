# Video Generation Prompts

视频生成阶段使用的固定 prompt，单套通用，适配所有角色气质。
每个状态备 3 条变体，生成多个视频时随机轮用。

---

## 固定前缀

所有 prompt 统一加上以下前缀：

```
固定镜头，保持人物形象一致性，背景不变，保持竖屏构图：
```

---

## Idle（待机）

```
固定镜头，保持人物形象一致性，背景不变，保持竖屏构图：人物放松地静止站立，目光柔和看向镜头，伴随自然的轻微呼吸起伏与小幅度肢体微动
```

```
固定镜头，保持人物形象一致性，背景不变，保持竖屏构图：人物安静站立，视线自然落向镜头，偶尔轻轻整理衣角、拂过发梢，整体无大幅度动作
```

```
固定镜头，保持人物形象一致性，背景不变，保持竖屏构图：人物自然站立，始终看向镜头，保持松弛状态，伴随缓慢的头部微晃与手部轻摆的待机小动作
```

---

## Talk（说话）

```
固定镜头，保持人物形象一致性，背景不变，保持竖屏构图：人物对着镜头持续说话，同步自然的说话口型，搭配符合日常对话的手部辅助动作
```

```
固定镜头，保持人物形象一致性，背景不变，保持竖屏构图：人物面向镜头生动说话，口型清晰贴合发音，伴随富有情绪的手势动作，神态鲜活有感染力
```

```
固定镜头，保持人物形象一致性，背景不变，保持竖屏构图：人物对着镜头温和说话，口型自然流畅，搭配轻柔的手部小动作，神态柔和舒缓
```

---

## Think（思考）

```
固定镜头，保持人物形象一致性，背景不变，保持竖屏构图：人物面向镜头做出思考神态，单手托腮，伴随自然的皱眉、歪头等思考相关肢体动作
```

```
固定镜头，保持人物形象一致性，背景不变，保持竖屏构图：人物对着镜头呈现思考状态，做出挠头、摸下巴的动作，神态带着疑惑与思索感
```

```
固定镜头，保持人物形象一致性，背景不变，保持竖屏构图：人物站立着看向镜头，歪头做出思考模样，伴随手部轻抵嘴唇的自然思考小动作
```

---

## Listen（聆听）

```
固定镜头，保持人物形象一致性，背景不变，保持竖屏构图：人物面向镜头做出认真聆听的表情，面部神态随聆听内容自然变动，保持专注姿态
```

```
固定镜头，保持人物形象一致性，背景不变，保持竖屏构图：人物对着镜头呈现好奇聆听的状态，微微前倾身体，面部表情生动变化，眼神专注灵动
```

```
固定镜头，保持人物形象一致性，背景不变，保持竖屏构图：人物安静站立着聆听，面向镜头，面部带着柔和的聆听神态，眼神温和，伴随轻微的点头回应动作
```
